ARC Integration API

Sole gateway to the ARC legacy charge management system
Status:Ready
arc-api.internal.eeoc.gov / docs
Mediation Endpoints
All routes require Managed Identity Bearer token. Rate limit: 120 req/min per caller.
ARC Integration API mediation endpoints showing HTTP method, path, description, and authentication requirement
Method Path Description Auth
GET /arc/v1/mediation/eligible-cases Watermark-paginated list of charges eligible for mediation. Polls every 15 min. Managed Identity
GET /arc/v1/mediation/cases/{charge_number} Full charge record including basis, parties, filing date, and investigation status. Managed Identity
POST /arc/v1/mediation/cases/{charge_number}/status Write mediation status (Accepted, Declined, Scheduled, Settled, Withdrawn) back to ARC. Managed Identity
PUT /arc/v1/mediation/cases/{charge_number}/staff Assign mediator. ARC staff ID validated against the active roster before write. Managed Identity
POST /arc/v1/mediation/cases/{charge_number}/close Record disposition and close charge. Service Bus event publishing planned. Managed Identity
POST /arc/v1/mediation/cases/{charge_number}/documents Attach a document reference in ARC. Blob URL stored; content remains in Azure Blob. Managed Identity

Key Features

  • SSRF prevention with DNS rebinding protection
  • Exponential backoff retry with jitter
  • Managed Identity — no stored credentials
  • Service Bus event publishing (planned)
  • RFC 7807 Problem Details on all errors

Infrastructure

FastAPI (Python 3.13) Azure App Service Azure Service Bus Redis (circuit breaker state)

Hard Constraint

No other platform application calls ARC directly. All charge data flows through this service. Violations are blocked at the network layer via Azure VNET routing.

Observability

  • X-Request-ID propagated through every ARC hop
  • Watermark logged per sync cycle to Azure Table
  • Circuit breaker state visible in health endpoint

Unified Data Analytics Platform (UDAP)

Cross-program analytics — AI assistant, Superset dashboards, JupyterHub notebooks
Status:In Progress

Natural Language Queries

Ask questions in plain English across charge data, mediation outcomes, investigation metrics, and staffing KPIs. Results are returned as tables, charts, or prose summaries.

Automated Narrative Generation

Quarterly and annual reports drafted from live data. Leadership-facing language with HMAC-signed audit records on every AI-generated paragraph.

Superset Dashboards

Interactive dashboards with charge volume trends, settlement rates by office, basis breakdowns, and time-to-resolution distributions. Drill-down by district, office, and time period.

JupyterHub Notebooks

Ad-hoc analysis environment for data staff. Pre-configured with EEOC data connectors, PII masking utilities, and shared kernel access across the analytics team.
