Dataset Detail

fct_charge_filings

Source: ARC  |  Updated: Daily  |  142,847 records

Columns

  • charge_number
  • filing_date
  • basis_code
  • office_id
  • respondent_name_hash
  • status

Sample Query

SELECT
  office_id,
  basis_code,
  COUNT(*) AS charge_count,
  MIN(filing_date) AS earliest,
  MAX(filing_date) AS latest
FROM analytics.fct_charge_filings
WHERE
  filing_date >= '2025-10-01'
  AND status != 'withdrawn'
GROUP BY
  office_id,
  basis_code
ORDER BY
  charge_count DESC
LIMIT 50;

Note: respondent_name_hash is a SHA-256 + Key Vault salt hash. The original respondent name is not stored in the analytics layer.
